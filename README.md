# HideAppKernel

HideAppKernel simple application allows you to hide applications on screen sharing in discord and other similar applications for screen capture (like OBS Studio) by injecting dll file.

The apps you select will appear as black rectangles or become completely transparent in the screen sharing.

HideAppKernel hides all windows of the selected application, and if you open a new application window, it will also be hidden in the screen sharing.

[HideAppKernelLib](https://github.com/flyen256/HideAppKernelLib)

# Usage

Just download release file and run HideAppKernel.exe. Select app will hide/show it. Hidden apps will marked with X.

> [!WARNING]
> DONT USE IT ON APPS WITH ANTI-CHEAT, YOU CAN GET BANNED.

If you got any visual artifacts just press enter it will be refresh menu.
